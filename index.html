Welcome to The SuperOS Project.

This is a ComputerCraft Os system for everyday and advanced users. A though still in the making there are meany features. this includes and Error log and image and program manning. And simple login and Password setup. With a desktop much like windows you can put files where every you like and change there appearances easily and efficiently.

Meany people ask me what this is used for. in general its for the average moded Minecraft player that wishes to do something with Computer Craft but does not what to get to involved with learning to code and move about the computer.

Insights

the main purpose of this Project is to get new coders to learn and explore under a easy and fun way. Its also to see what the community can teach me about my programming habits and other things that i would not know with out them.
